# Angular2 サンプル

##. Node.js Install
 
https://nodejs.org/en/download/current/ から適切なインストーラーをダウンロードして適当に展開、Path を通しておく。

##. 実行する

```
$ npm install
$ npm start
```
##. index を開く

http://localhost:3000/
